# BizMargin Changelog

## v0.1.0 — 2026-06-03
### MVP release
- 5 calculators: Profit Margin, Markup, Selling Price, Break-Even, Amazon FBA
- Amazon FBA category presets (Electronics 8%, Books 6%, Clothing 17% etc.)
- Marketplace presets (US, UK, EU, CA)
- FBA size presets (Small std, Large std, Oversize)
- localStorage save/restore — namespace bizmargin.v1, 25-item LRU
- Intl.NumberFormat locale-correct currency formatting
- URL hash permalink for shareable scenarios
- CSV export for FBA results
- GDPR cookie consent — AdSense lazy-loads after accept only
- 80ms debounce on all inputs
- ARIA labels and aria-live regions
- Input validation with inline error messages
- Unprofitable warnings for negative margins
- Step +/- buttons on price inputs
- Cross-links to BizInvoiceGen and BizQRGen
- Hostinger affiliate link
