# BizMargin

Free profit margin calculator for Amazon FBA sellers, dropshippers and small businesses.

**Live:** https://bizmargin.com

## Calculators
- Profit Margin
- Markup
- Selling Price
- Break-Even
- Amazon FBA (with category + marketplace presets)

## Stack
Single-file vanilla HTML/CSS/JS. No dependencies. No build step.

## Deploy
Push to `main` — Hostinger auto-deploys via Git integration.
